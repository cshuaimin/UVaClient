# uva-cli

A productive cli tool to enjoy [UVa Online Judge](https://uva.onlinejudge.org)!

- A very effficient way to fight questions.
- Test the code locally,
- or submit it to the UVa.

## Quick start

- Login with your UVa account:

  ```sh
  $ uva user -l
  ```

- Show description:

  ```sh
  $ uva show <ID>
  ```

- Run tests:

  ```sh
  $ uva test <file>
  ```

- Submit it!

  ```sh
  $ uva submit <file>
  ```
